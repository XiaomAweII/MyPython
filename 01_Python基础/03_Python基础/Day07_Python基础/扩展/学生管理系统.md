#### 学生管理系统

* 需求

  ```python
  1. 先打印提示界面(1-6的数字), 让用户选择他/她要进行的操作.
  2. 当用户选择1的时候, 实现操作: 添加学生(学生编号, 学生姓名, 手机号).
  3. 当用户选择2的时候, 实现操作: 删除学生(根据编号删除)
  4. 当用户选择3的时候, 实现操作: 修改学生信息(只能改姓名, 手机号)
  5. 当用户选择4的时候, 实现操作: 查询单个学生信息(根据姓名查)
  6. 当用户选择5的时候, 实现操作: 查询所有学生信息
  7. 当用户选择6的时候, 实现操作: 退出系统
  ```

* 目的

  把之前所需的知识点: **if, for, 函数等知识点**结合到一起, 做一个综合案例.

* 思路

  * 定义函数 print_info()  , 打印提示信息.
  * 自定义while True循环逻辑, 实现: 用户录入什么数据, 就进行相应的操作
    * 注意: 处理一下非法值.
  * 自定义函数 add_info(), 实现: 添加学生
    * 编号必须唯一
  * 自定义函数 delete_info(), 实现: 删除学生
    * 根据编号删除
  * 自定义函数 update_info(), 实现: 修改学生信息
    * 根据编号修改, 只能修改: 姓名, 手机号.
  * 自定义函数 search_info(), 实现: 查询某个学生信息.
    * 根据姓名查询
  * 自定义函数 search_all(), 实现: 查询所有学生的信息.

* 代码实现

  ```python
  '''
      需求:
          1. 先打印提示界面(1-6的数字), 让用户选择他/她要进行的操作.
          2. 当用户选择1的时候, 实现操作: 添加学生(学生编号, 学生姓名, 手机号).
          3. 当用户选择2的时候, 实现操作: 删除学生(根据编号删除)
          4. 当用户选择3的时候, 实现操作: 修改学生信息(只能改姓名, 手机号)
          5. 当用户选择4的时候, 实现操作: 查询单个学生信息(根据姓名查)
          6. 当用户选择5的时候, 实现操作: 查询所有学生信息
          7. 当用户选择6的时候, 实现操作: 退出系统
      步骤/思路:
          1. 定义函数 print_info()  , 打印提示信息.
          2. 自定义while True循环逻辑, 实现: 用户录入什么数据, 就进行相应的操作
            注意: 处理一下非法值.
          3. 自定义函数 add_info(), 实现: 添加学生
            编号必须唯一
          4. 自定义函数 delete_info(), 实现: 删除学生
            根据编号删除
          5. 自定义函数 update_info(), 实现: 修改学生信息
            根据编号修改, 只能修改: 姓名, 手机号.
          6. 自定义函数 search_info(), 实现: 查询某个学生信息.
            根据姓名查询
          7. 自定义函数 search_all(), 实现: 查询所有学生的信息.
  '''
  
  
  # 1. 定义函数 print_info()  , 打印提示信息.
  def print_info():
      print('1. 添加学生')
      print('2. 删除学生')
      print('3. 修改学生信息')
      print('4. 查询单个学生信息')
      print('5. 查询所有的学生信息')
      print('6. 退出系统')
  
  
  # 3. 自定义函数 add_info(), 实现: 添加学生,  编号必须唯一
  # 3.1 定义全局变量, 用来记录所有学生的信息, 其实是一个: 列表嵌套字典的形式.
  # 例如: 数据格式   [{'id':'heima001', 'name':'刘亦菲', 'tel': '111'}, {'id':'heima002', 'name':'高圆圆', 'tel': '222'}]
  info = []
  
  
  # 3.2 定义函数, 实现添加学生.
  def add_info():
      # 3.3 提示用户录入学生信息并接收.
      new_id = input('请录入您要添加的学生的编号: ')
  
      # 3.4 遍历学生列表, 获取到每个学生信息.
      global info
      for stu in info:
          # 3.5 如果当前用户录入的编号已经存在, 则: 不添加该学生.
          if stu['id'] == new_id:
              print('您录入的学生编号已存在, 请重新操作')
              return  # 结束函数
  
      new_name = input('请录入您要添加的学生的姓名: ')
      new_tel = input('请录入您要添加的学生的手机号: ')
  
      # 3.6 将用户录入的学生信息, 封装成: 字典形式.
      dict_info = {'id': new_id, 'name': new_name, 'tel': new_tel}
      # 3.7 将封装后的学生信息, 添加到: 学生列表中
      info.append(dict_info)
  
  
  # 4. 自定义函数 delete_info(), 实现: 删除学生, 根据编号删除
  # 4.1 定义函数 delete_info()
  def delete_info():
      # 4.2 提示用户录入要删除的学生编号, 并接收.
      del_id = input('请录入您要删除的学生的编号: ')
      # 4.3 遍历学生列表, 获取到每个学生信息.
      global info
      for stu in info:
          # 4.4 判断当前用户录入的编号是否存在, 存在就删除该学生.
          if stu['id'] == del_id:
              info.remove(stu)
              print('删除成功!')
              break  # 结束循环
      else:
          # 4.5 如果录入的编号不存在, 则不删除, 并提示.
          print('您录入的学生编号不存在, 请重新操作')
  
  
  # 5. 自定义函数 update_info(), 实现: 修改学生信息, 根据编号修改, 只能修改: 姓名, 手机号.
  # 5.1 定义函数, 修改学生信息.
  def update_info():
      # 5.2 提示用户录入要修改的学生编号, 并接收.
      upt_id = input('请录入您要修改的学生的编号: ')
      # 5.3 遍历学生列表, 获取到每个学生信息.
      global info
      for stu in info:
          # 5.4 判断当前用户录入的编号是否存在, 存在就修改该学生信息.
          if stu['id'] == upt_id:
              # 5.4.1 提示用户录入新的姓名和手机号
              new_name = input('请录入修改后的学生的姓名: ')
              new_tel = input('请录入修改后的学生的手机号: ')
  
              # 5.4.2 根据用户录入的信息, 修改上述的数据.
              # stu学生信息格式:   {'id': new_id, 'name': new_name, 'tel': new_tel}
              stu['name'] = new_name
              stu['tel'] = new_tel
              print('修改成功!')
              break  # 结束循环
      else:
          # 5.5 如果录入的编号不存在, 则不修改, 并提示.
          print('您录入的学生编号不存在, 请重新操作')
  
  
  # 6. 自定义函数 search_info(), 实现: 查询某个学生信息, 根据姓名查询
  # 6.1 定义函数 search_info(), 查询单个学生信息
  def search_info():
      # 6.2 提示用户录入要查询的学生姓名, 并接收.
      search_name = input('请录入您要查询的学生的姓名: ')
  
      # 6.3 遍历学生列表, 获取到每个学生信息.
      # 定义flag变量, 记录是否查询到了学生.
      flag = False  # 默认没有查询到.
      global info
      for stu in info:
          # 6.4 判断当前用户录入的姓名是否存在, 存在就打印该学生的信息
          if stu['name'] == search_name:
              flag = True  # 说明查询到学生了
              print(f'学生编号: {stu["id"]}, 姓名: {stu["name"]}, 手机号: {stu["tel"]}')
  
      # 6.5 如果录入的姓名不存在, 则不打印, 并提示.
      if flag == False:
          print('您录入的学生姓名不存在, 请重新操作')
  
  
  # 7. 自定义函数 search_all(), 实现: 查询所有学生的信息.
  # 7.1 定义函数 search_all(), 查询所有信息.
  def search_all():
      # 7.2 直接遍历学生列表即可.
      global info
      for stu in info:
           print(f'学生编号: {stu["id"]}, 姓名: {stu["name"]}, 手机号: {stu["tel"]}')
  
  
  
  # 2. 自定义while True循环逻辑, 实现: 用户录入什么数据, 就进行相应的操作
  # 2.1 死循环实现该需求, 只要用户不选择 退出系统, 就让用户一直玩儿.
  while True:
      # 细节: 调用 print_info()函数, 打印: 操作界面
      print_info()
  
      # 2.2 提示用户录入要进行的操作, 并接收.
      num = input('请录入您要操作的编号: ')
      # 2.3 根据用户录入的编号, 进行相应的操作.
      if num == '1':
          # 3.8 在这里调用 添加学生的函数
          add_info()
      elif num == '2':
          # 4.6 调用函数, 删除学生
          delete_info()
      elif num == '3':
          # 5.6 调用函数, 修改学生信息
          update_info()
      elif num == '4':
          # 6.6 调用函数, 查询
          search_info()
      elif num == '5':
          # 7.2 调用函数, 查询所有信息.
          search_all()
      elif num == '6':
          print('谢谢您使用, 欢迎下次光临!')
          break
      # 细节: 处理一下非法值
      else:
          print('录入有误, 请重新录入:\n')
  
      print('=====按Enter键继续操作=====\n')
  ```
